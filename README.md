# 📚 Bootcamp DevJr — Módulo 8: Car System 

Aplicação frontend para gerenciamento de carros desenvolvida como desafio do Bootcamp DevJr.

## 🖥️ Funcionalidades:

- Cadastro de carros 
- Listagem de carros em tabela
- Edição e remoção de carros cadastrados

## 🛠️ Tecnologias: 

- Angular 21
- Bootstrap 
- TypeScript

## Como rodar
Clone o repositório e instale as dependências:

```bash
git clone https://github.com/Kauan-Berti/bootcamp-devjr-modulo8-car-system.git

cd bootcamp-devjr-modulo8-car-system

npm install
```

Inicie o servidor de desenvolvimento:

```bash
ng serve
```

Acesse `http://localhost:4200` no navegador.
